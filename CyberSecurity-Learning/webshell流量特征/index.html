webshell流量特征

2023/4/25 CyberSecurity Learning pt

webshell工具流量特征

菜刀

  • 老版采用明文传输
  • 新版采用base64加密，发现大量base64加密密文就要注意
  • 使用HTTP协议通信
  • 请求头部包含“User-Agent: Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.1)”
  • 请求数据包中包含cmd.exe、regedit.exe等特殊命令

蚁剑

  • 部分代码明文传输，较好辨认

    ini_set
ini_set_time
ini_set_limit
@ini_set(“display_errors”,“0”)

  • 使用HTTP或HTTPS协议通信

  • 请求头部包含“User-Agent: Mozilla/5.0 (Windows NT 10.0; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/58.0.3029.110 Safari/537.36”

  • 请求数据包中包含cmd.exe、powershell.exe、cat等特殊命令

哥斯拉

  • 正常请求cookie最后结尾没有分号
  • 会相应三次
  • connection : keep-alive
  • 使用TCP协议进行通信
  • 请求数据包中包含/bin/bash、/usr/bin/python等特殊命令

冰蝎

  • 与C2服务器的通信采用随机化的URI，通常采用混淆技术来欺骗网络安全防护
  • 可以使用多种通信协议，包括HTTP和HTTPS等
  • 可以通过DNS隧道或HTTP隧道等技术来绕过网络防御
